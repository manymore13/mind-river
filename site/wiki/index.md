# 知识库

这里是 LLM 维护的知识沉淀层，包含概念、主题和笔记三类内容。

## 分类

- **概念** ([/wiki/concepts/](/wiki/concepts/))：定义、术语、基础知识
- **主题** ([/wiki/topics/](/wiki/topics/))：特定主题的深入探讨
- **笔记** ([/wiki/notes/](/wiki/notes/))：阅读笔记、学习心得

## 特点

- 由 LLM 自动维护和更新
- 与原始资料关联，支持双向跳转
- 交叉引用，形成知识网络
