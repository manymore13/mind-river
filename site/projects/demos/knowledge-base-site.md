# 个人知识库网站

这是我正在开发的个人知识库网站项目。

## 项目简介

基于 VitePress 构建的个人知识管理系统，融合 LLM Wiki 设计思想。

## 核心功能

- 三层架构：原始资源层、知识沉淀层、配置层
- 双版本关联：原始资料与 Wiki 解读版互相链接
- 自定义导航：灵活的导航栏和侧边栏配置
- 广告支持：预留广告位
- 评论系统：基于 Giscus

## 技术栈

```
- VitePress 1.x
- Vue 3
- TypeScript
- Giscus
```
