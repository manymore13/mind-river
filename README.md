# Personal Knowledge Base

A personal website for knowledge management, built with VitePress.

## Structure

```
├── raw/           # Original resources (immutable)
├── site/          # Website source code
│   ├── wiki/      # Knowledge base (LLM maintained)
│   ├── blog/      # Blog posts
│   └── projects/  # Project showcases
├── README.md
└── LICENSE
```

## Development

```bash
cd site
npm install
npm run dev
```

## Build

```bash
cd site
npm run build
```
